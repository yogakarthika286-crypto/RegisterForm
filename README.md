# Registration Form

A responsive Registration Form created using HTML5 and CSS3. This project is designed to collect basic user information through a clean and user-friendly interface.

## Features

- Full Name input
- Email Address field
- Mobile Number field
- Password field
- Date of Birth (DOB) picker
- Address (Textarea)
- Country (Dropdown)
- Gender (Radio Buttons)
- Agree Condition (Checkbox Buttons)
- Submit and Reset Buttons
- Clean and responsive design

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created to practice HTML form elements, CSS styling, form alignment, and responsive web design.

## Future Improvements

- Add JavaScript form validation
- Connect the form with a backend database
- Store user data securely
